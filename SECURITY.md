# Security Policy

## Scope

This security policy covers:

- **Macro-generated code safety** — Patterns in code generated by `@ChatCompletionsToolArguments` and `@ChatCompletionsTool` macros that could introduce vulnerabilities
- **Dependency vulnerabilities** — Issues in swift-syntax or other dependencies
- **JSON Schema injection** — Crafted inputs that could produce malicious schema output

## Reporting a Vulnerability

1. Open a GitHub Issue with the `security` label
2. Describe the vulnerability at a high level — **do not include exploit details** in the public issue
3. Request private communication for sharing specifics

## Response

- **Acknowledgment**: Within 48 hours of the report
- **Assessment**: Severity and fix timeline communicated within 7 days
- **Fix**: Security patches prioritized over feature work

## Supported Versions

Only the latest released version receives security updates.
